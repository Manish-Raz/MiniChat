# 💬 Mini Chat Application

A **real-time Mini Chat Application** built with **Node.js, Express.js, JavaScript, and MongoDB**.
This project demonstrates a simple full-stack chat system where users can send and receive messages through a web interface with messages stored in a database.

# ⚙️ Installation & Setup

## 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

## 2️⃣ Navigate to the project folder

```bash
cd your-repo-name
```

## 3️⃣ Install dependencies

```bash
npm install
```

---

# 🗄️ MongoDB Setup

Make sure **MongoDB is installed and running locally** or use **MongoDB Atlas**.

Example connection string:

```
mongodb://127.0.0.1:27017/chatDB
```

Update the database connection inside `index.js`.

---

# ▶️ Run the Application

```bash
node index.js
```

or using **nodemon**

```bash
npx nodemon index.js
```

Now open:

```
http://localhost:3000
```
